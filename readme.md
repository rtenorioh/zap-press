### Como usar

- Clone ou faça download deste repositório
- Entre na pasta do projeto
- Rode o comando `npm install`
- Depois `npm start`
- Com o projeto rodando, acessar o browser no endereço `http://localhost:8000`
- Ecaneio o QR Code
- Pronto para uso!
